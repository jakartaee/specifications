HTML_BASIC render-kit
component-family: javax.faces.Script renderer-type: javax.faces.Websocket

Render a JavaScript function that invokes jsf.push.init() function.

Encode Behavior

Render an HTML script element with an "id" attribute. Render JavaScript function which invokes jsf.push.init() function with data provided by onopen, onmessage, onclose and connected arguments, if any.

This renderer is not responsible for rendering its children.

This renderer-type has no attributes
