The Jakarta EE Core Profile defines a profile of the Jakarta EE Platform targeted at cloud native microservice
applications with support for build time/ahead of time compilation.

* [Jakarta Core Profile 10 Release Record](https://projects.eclipse.org/projects/ee4j.jakartaee-platform/releases/core-profile-10)
